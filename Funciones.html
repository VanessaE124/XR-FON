XR-FON

  • Somos una librería de realidad extendida para dispositivos con sistema operativo Android. Permite el seguimiento de un objeto físico fijo (mesa), de una dimensión variable de 20 cm^2 a 2 m^2, y dos objetos físicos rígidos que pueden ser ubicados sobre la mesa; y que pueden ser manipulados en el entorno virtual por el usuario.

    • Funciones

  • Funcionar en dispositivos móviles tipo Android
  • Está desarrollado para el motor de videojuegos Unity
  • Visualización de un entorno virtual
  • Interacción de objetos del mundo real en el entorno virtual

    • Prerrequisitos

  • Instala Unity
  • Utiliza Unity Hub preferiblemente
  • Instala Vuforia en Unity
  • Te recomendamos las versiones desde el 2020 en adelante
  • Su computador debe disponer de cámara para la compilación de la escena.

    Script ButtonManager

    Se crean dos UnityEvent, OnButtonPressed y OnButtonReleased, los cuales serán los eventos que almacenarán las acciones al momento de oprimir el botón; además del Gameobject, Habitación, que sirve para enlazar el modelo del entorno al virtual button. Por consiguiente, se crea la función private void OnEnable que registra cuando el botón es presionado y cuando no, a su vez, dentro de esta, se desactiva el GameObject para que solo se muestre el entorno virtual en caso de que llegue a ser oprimido el botón.

    En la función ButtonPressed, se llama al evento OnButtonPressed, que, mediante un GameObject.SetActive, activa la habitación; por último, se agrega un Debug.Log para que se envíe un mensaje en caso de que el botón sea presionado.

    Por su parte, en la función ButtonReleased, se activa el UnityEvent OnButtonReleased, el cual tiene los mismos parámetros que el OnButtonPressed; este se encuentra disponible por si el usuario desea agregar una sección o función al evento. Para finalizar se crea la función OnDestroy, que sirve para remover las instancias del ButtonPressed en el caso de que se active el ButtonReleased y viceversa.

    Script posición de la mesa

    Este script se realiza para establecer una mesa de trabajo de tamaño variable que depende de dos targets.

    1. En la primera parte se hace la declaración de variables y vectores necesarios para el desarrollo
    2. Establecer los dos targets, para esto debe ser declarado tanto en el script como en la ventana de ejecución de Unity
    3. Luego procedemos a realizar las clases para el cálculo de posiciones
    4. Obtener posición: permite extraer la posición de los targets para posteriormente hacer el llamada a dos funciones
    5. Calcula la posición: permite hacer las operaciones necesaria en X y Z por medio de la información entrada de uno de los targets
    6. Calculo escala: la función toma las coordenadas de segundo targets para establecer el tamaño de la mesa
    7. SeteoPosicion: la función permite establecer la mesa en las posiciones extraídas y calculadas anteriormente
    8. SeteoEscala: establece la escala de la mesa para dejarla finalmente en una posición correcta
