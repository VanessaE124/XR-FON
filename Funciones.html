XR-FON

Somos una librería de realidad extendida para dispositivos con sistema operativo Android. Permite el seguimiento de un objeto físico fijo (mesa), de una dimensión variable de 20 cm^2 a 2 m^2, y dos objetos físicos rígidos que pueden ser ubicados sobre la mesa; y que pueden ser manipulados en el entorno virtual por el usuario.

VUFORIA

Vuforia es una plataforma de desarrollo de aplicaciones de Realidad Aumentada (AR) y Realidad Mixta (MR) multiplataforma

Uso de targets

Se utilizan imágenes que Vuforia detecta y rastrea para mostrar objetos físicos en el entorno virtual.

Model target generator

Se implementa para capturar la mesa del entorno físico y ponerlo en el entorno virtual.
